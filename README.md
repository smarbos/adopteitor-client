# adopteitor-client

### Instalacion

#### Client:

Install NPM:

    nmp install

Start NPM

    npm start

## Installing MysqlDB

I use:
sudo apt-get install libmysqlclient-dev

Then i run:
apt-get install python-dev

Finnaly:
pip install MySQL-python

Works!


### Usage

Run client:
    cd client && npm start

Run server:
    cd server && workon venv && python manage.py runserver

Run gulp:
    cd client && gulp && gulp watch
