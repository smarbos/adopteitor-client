# adopteitor-client

### Instalacion

#### Client:

Install NPM:

    nmp install

Install Bower:

    bower install


## Installing MysqlDB

I use:
sudo apt-get install libmysqlclient-dev

Then i run:
apt-get install python-dev

Finnaly:
pip install MySQL-python

Works!


### Usage

Run client:
    cd adopteitor-client && gulp
